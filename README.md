# Python To-Do Task Manager

A To-Do Task Manager application built using Python.

This repository contains two versions of the project:

## 1. CLI Version
Features:
- Add Tasks
- View Tasks
- Update Tasks
- Delete Tasks
- Mark Tasks as Done
- Multiple Task Deletion
- File Handling

Storage File:
- tasks.txt

## 2. Tkinter GUI Version
Features:
- Add Tasks
- View Tasks
- Update Tasks
- Delete Tasks
- Mark Tasks as Done
- File Handling
- Graphical User Interface

Storage File:
- tasks_UI.txt

## Technologies Used
- Python
- Tkinter
- File Handling

## Learning Outcomes
- CRUD Operations
- File Handling
- Event-Driven Programming
- GUI Development with Tkinter
- Input Validation
- Debugging and Problem Solving
